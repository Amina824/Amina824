### 🌱 AI-Driven GEC Management System

An AI-powered platform for managing the lifecycle of **Granular Energy Certificates**, including certificate creation, transfer, retirement, search, and marketplace operations.

**Key Highlights**

- Developed a conversational AI chatbot that allows users to perform certificate-related tasks using natural language
- Integrated a **RAG pipeline** to retrieve domain-specific certificate information and improve response accuracy
- Designed **agentic workflows** using AI agents and Model Context Protocol for task automation
- Built REST APIs for certificate lifecycle management and marketplace operations
- Designed a decentralized system approach for transparent certificate trading and ownership tracking
- Focused on making blockchain-based operations easier for non-technical users

**Tech Stack:** Generative AI, LangChain, RAG, AI Agents, MCP, REST APIs, Decentralized Systems

### 🧪 Society Registration System Simulation & Research

A simulation-based research project focused on improving the student society registration process by analyzing workflow inefficiencies and optimizing approval steps.

**Key Highlights**

- Analyzed data from **1000+ students** to identify bottlenecks in the registration process
- Modeled and compared manual, digital, and AI-driven registration workflows
- Reduced simulated registration time from **20 minutes to 1 minute**
- Designed an optimized workflow with automated validation and approval logic

**Tech Stack:** AnyLogic, Simulation Modeling, Process Engineering, Data Analysis
